# Golden Prompts - Deep News OAI

ChatGPT에서 도구 호출 품질을 테스트하기 위한 골든 프롬프트 모음입니다.

## 테스트 방법

1. ChatGPT Desktop에서 MCP 서버 연결 확인
2. 각 프롬프트를 입력하고 예상 동작 확인
3. 도구 호출 로그에서 호출된 도구 확인

---

## 1. search_korean_news (뉴스 검색)

### Direct (정확히 호출되어야 함)

| 프롬프트 | 예상 파라미터 |
|---------|-------------|
| "AI 관련 최근 한국 뉴스 검색해줘" | keyword: "AI", 최근 날짜 |
| "삼성전자 관련 기사 찾아줘" | keyword: "삼성전자" |
| "2025년 1월 경제 뉴스 보여줘" | keyword: "경제", start_date: "2025-01-01" |
| "반도체 뉴스 10개만 검색" | keyword: "반도체", page_size: 10 |
| "지난주 정치 관련 기사 검색" | keyword: "정치", 최근 7일 |

### Indirect (문맥상 호출 가능)

| 프롬프트 | 예상 동작 |
|---------|----------|
| "요즘 반도체 산업 어때?" | search_korean_news("반도체") 호출 가능 |
| "최근 한국 경제 상황이 궁금해" | search_korean_news("경제") 호출 가능 |
| "테슬라 관련 한국 보도 어떻게 되고 있어?" | search_korean_news("테슬라") 호출 |
| "요즘 취업 시장 뉴스 좀 알려줘" | search_korean_news("취업") 호출 |
| "최근 환율 관련 기사 있어?" | search_korean_news("환율") 호출 |

### Negative (호출되면 안 됨)

| 프롬프트 | 이유 |
|---------|-----|
| "미국 날씨 어때?" | 한국 뉴스와 무관 |
| "파이썬 코드 작성해줘" | 코딩 요청 |
| "영어로 번역해줘" | 번역 요청 |
| "오늘 점심 뭐 먹을까?" | 일상 대화 |
| "유럽 축구 경기 결과 알려줘" | 외국 스포츠 (한국 언론 미보도) |

## 2. count_news_articles (기사 수 조회)

### Direct

| 프롬프트 | 예상 파라미터 |
|---------|-------------|
| "AI 관련 기사가 몇 개 있어?" | keyword: "AI" |
| "지난달 삼성 뉴스 몇 건?" | keyword: "삼성", 지난달 날짜 |
| "올해 반도체 기사 총 개수" | keyword: "반도체", 2025년 전체 |
| "지난주 정치 기사 수 알려줘" | keyword: "정치", 최근 7일 |
| "2025년 1월 경제 뉴스 몇 건이야?" | keyword: "경제", 1월 날짜 범위 |

### Indirect

| 프롬프트 | 예상 동작 |
|---------|----------|
| "요즘 AI 뉴스 많이 나와?" | count_news_articles 또는 search 호출 |
| "반도체 기사가 얼마나 쏟아지고 있어?" | count_news_articles 호출 가능 |
| "최근 취업 관련 보도량이 궁금해" | count_news_articles 호출 |

### Negative

| 프롬프트 | 이유 |
|---------|-----|
| "AI가 뭐야?" | 정의 질문, 검색 아님 |
| "내 문서 몇 개야?" | 개인 파일, 뉴스 아님 |

## 3. get_korean_time (한국 시간)

### Direct

| 프롬프트 | 예상 결과 |
|---------|----------|
| "지금 한국 시간이 몇 시야?" | 현재 KST 반환 |
| "한국 현재 시각" | 현재 KST 반환 |
| "오늘 날짜가 뭐야?" | 현재 날짜 반환 |
| "KST 시간 알려줘" | 현재 KST 반환 |
| "한국 시간으로 지금 몇 시?" | 현재 KST 반환 |

### Indirect

| 프롬프트 | 예상 동작 |
|---------|----------|
| "오늘 뉴스 검색하려면 날짜가..." | get_korean_time 먼저 호출 가능 |
| "지금 서울은 몇 시쯤 돼?" | get_korean_time 호출 |

### Negative

| 프롬프트 | 이유 |
|---------|-----|
| "뉴욕 시간 알려줘" | 한국 시간 아님 |
| "UTC 시간이 뭐야?" | KST 전용 도구 |
| "일본 시간은?" | 한국 시간 아님 |

## 4. list_news_providers (언론사 목록)

### Direct

| 프롬프트 | 예상 결과 |
|---------|----------|
| "한국 언론사 목록 보여줘" | 언론사 리스트 반환 |
| "어떤 신문사들이 있어?" | 언론사 리스트 반환 |
| "뉴스 제공 언론사가 뭐가 있어?" | 언론사 리스트 반환 |
| "경제지 목록 알려줘" | 언론사 리스트 (경제지 포함) |
| "방송사 뉴스 어디서 가져와?" | 언론사 리스트 반환 |

### Indirect

| 프롬프트 | 예상 동작 |
|---------|----------|
| "조선일보는 이 서비스에서 검색 가능해?" | list_news_providers 호출 가능 |

### Negative

| 프롬프트 | 이유 |
|---------|-----|
| "미국 뉴스 사이트 추천해줘" | 한국 언론사 아님 |
| "BBC 기사 검색해줘" | 외국 언론, 목록 조회 아님 |

## 5. find_news_category (카테고리 검색)

### Direct

| 프롬프트 | 예상 결과 |
|---------|----------|
| "정치 카테고리 코드가 뭐야?" | 정치 카테고리 정보 |
| "경제 뉴스 분류 코드 알려줘" | 경제 카테고리 정보 |
| "스포츠 섹션 코드" | 스포츠 카테고리 정보 |
| "IT 카테고리 찾아줘" | IT_과학 카테고리 정보 |
| "사회 뉴스 분류 번호" | 사회 카테고리 정보 |

### Indirect

| 프롬프트 | 예상 동작 |
|---------|----------|
| "뉴스 카테고리가 어떻게 나뉘어져 있어?" | find_news_category 또는 전체 목록 |

### Negative

| 프롬프트 | 이유 |
|---------|-----|
| "정치 뉴스 검색해줘" | 검색이지 카테고리 조회 아님 |
| "경제 상황 어때?" | 뉴스 검색 요청 |

## 테스트 체크리스트

### 연결 테스트
- [ ] ChatGPT Desktop → Settings → Developer Tools
- [ ] MCP Servers → URL: `https://deepnews-oai.seolcoding.com/sse`
- [ ] Refresh 클릭 → 도구 목록 5개 확인

### 도구 호출 테스트
- [ ] "AI 뉴스 검색해줘" → search_korean_news 호출
- [ ] "지금 한국 시간" → get_korean_time 호출
- [ ] "언론사 목록" → list_news_providers 호출
- [ ] "AI 기사 몇 개?" → count_news_articles 호출
- [ ] "정치 카테고리 코드" → find_news_category 호출

### Widget 렌더링 테스트
- [ ] 검색 결과 → 카드 UI 표시 확인
- [ ] 기사 클릭 → 상세 보기 동작 확인
- [ ] 다크/라이트 테마 전환 확인

### 에러 처리 테스트
- [ ] 잘못된 날짜 입력 → 에러 메시지 확인
- [ ] 검색 결과 없음 → 적절한 메시지 확인

### 부정 테스트 (호출 안 되어야 함)
- [ ] "미국 날씨 알려줘" → search_korean_news 미호출
- [ ] "코드 작성해줘" → 어떤 도구도 미호출
